# Quartas de Programação - Pedidos de Cafeteria (Aula)

Projeto em Node.js com estrutura simples de `src/` e `test/`.

## Instalação

```bash
npm install
```

## Testes

Os testes são configurados para rodar com o Mocha.

Para rodar os testes:

```bash
npm test
```

## Estrutura

- `src/` - código fonte do projeto
- `test/` - testes

## Observações

- Adicione seus scripts de execução e testes conforme o projeto crescer.
