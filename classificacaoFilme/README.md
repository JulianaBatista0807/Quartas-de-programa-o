# Quartas de Programação

Projeto em Node.js com estrutura simples de `src/` e `test/`.

## Instalação

```bash
npm install
```

## Execução

Adapte conforme necessário. Por padrão, o `package.json` não define script de execução específico.

## Testes

Atualmente não há testes configurados no `package.json`.

Para rodar os testes quando estiverem prontos:

```bash
npm test
```

## Estrutura

- `src/` - código fonte do projeto
- `test/` - testes

## Observações

- O projeto usa `type": "module"` no `package.json`, então arquivos ES Modules são suportados.
- Adicione seus scripts de execução e testes conforme o projeto crescer.
