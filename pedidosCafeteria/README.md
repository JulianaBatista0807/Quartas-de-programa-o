# Quartas de Programação - Pedidos de Cafeteria - feito por mim

Projeto em Node.js com estrutura simples de `src/` e `test/`.

## Instalação

```bash
npm install
```

## Testes

Os testes são configurados para rodar com o Mocha e gerar um relatório com o Mochawesome.

Para rodar os testes:

```bash
npm test
```

## Estrutura

- `src/` - código fonte do projeto
- `test/` - testes

## Observações

- Adicione seus scripts de execução e testes conforme o projeto crescer.
